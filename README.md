## What is Django?
- Python web framework
- Server Side Framework
- Designed to get you started fast
- Batteries included
- MVP Design Pattern

## What is a Web Framework?
- Collection of modules, packages and libraries
- Designed to speed up development

**Frontend (Client Side)**
